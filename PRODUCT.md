# Product

## Register

product

## Users

Solana developers integrating token vesting into their projects. Context: evaluating whether Velora fits their needs, then building against the SDK. Secondary: token creators configuring vesting schedules through the UI. Technical audience comfortable with Anchor, PDAs, and SPL tokens.

## Product Purpose

Velora is an on-chain token vesting protocol on Solana. It lets creators lock SPL tokens into program-controlled vaults and release them to recipients on configurable schedules (linear, cliff, cliff+linear, milestone). The docs site is the primary surface for developers to understand, integrate, and troubleshoot the protocol.

## Brand Personality

Precise, technical, trustworthy. Voice is direct and reference-oriented, not marketing-forward. Three words: **clear, confident, complete**. Developers should feel they found the authoritative source, not a landing page.

## Anti-references

- Overly marketing-heavy crypto docs (hype language, token price focus, "revolutionary" claims)
- Docs that bury API reference under narrative prose
- Generic AI-generated doc layouts (identical card grids, gradient backgrounds, glassmorphism)
- Sites that require account creation to read docs

## Design Principles

1. **Content earns attention.** Structure and hierarchy guide the eye, not decoration.
2. **Bittensor-caliber organization.** Card-based discoverability on the homepage, persistent sidebar for depth, search-first navigation.
3. **Velora brand consistency.** Dark theme, gold accent, Velora Geist typography carried from the main app into docs.
4. **Developer-first ergonomics.** Code blocks with syntax highlighting, copy buttons, clear API reference tables, fast search.
5. **Progressive disclosure.** Getting started on the surface, deep reference one click away.

## Accessibility & Inclusion

WCAG 2.1 AA. Keyboard-navigable sidebar and search. Code blocks with sufficient contrast. Reduced-motion support (no decorative animations in docs). Screen-reader friendly navigation landmarks.
